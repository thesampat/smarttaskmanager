# Smart Task Manager

A simple task manager built with the MERN stack and AI insights.

AI Tools Used
Gemini (Antigravity Assistant): Used to build the entire project, including the backend API, the React UI, and the AI task analysis.

1. Add your `GEMINI_API_KEY` to `backend/.env`.
2. Run with Docker:
   ```bash
   docker compose up --build
   ```

## Features
- AI-generated categories and difficulty levels.
- Premium light UI with Tailwind CSS.
- Fast caching with Redis.
